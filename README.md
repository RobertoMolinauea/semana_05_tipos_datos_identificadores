# Tarea – Tipos de datos e Identificadores

Este repositorio contiene un programa básico en Python desarrollado como parte de la asignatura
Programación Orientada a Objetos.

## Descripción
El programa calcula el total a pagar de un producto, aplicando descuento e IVA.

## Tipos de datos utilizados
- int
- float
- string
- boolean

## Convenciones
- Uso de snake_case en variables y funciones
- Identificadores claros y descriptivos
